# SmartQueue Frontend

Real-time queue management system built with React + Vite.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Features

- Real-time queue updates via WebSocket
- User & Admin authentication
- Category-based place listings
- Region-based filtering
- Slot booking system
- Smart reminders
- QR code tickets

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Socket.io Client
- Framer Motion
- React Router v6
